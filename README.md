# 📱 Zimbra Mobile

A React Native mobile application for Zimbra login and related functionality.

---

## 🚀 Getting Started

### 📦 Install Dependencies

```bash
npm install
```

---

## ▶️ Run the App

### 🤖 Android

```bash
npm run android
```

### 🍎 iOS

```bash
npm run ios
```

---

## ⚙️ Requirements

- Node & Npm
- Ruby
- OpenJDK
- Postman
- Homebrew
- Cocoapods
- Android Studio (for Android)
- Xcode (for iOS)
